## URL Shortener

This is a URL shortener application built with Express and MongoDB.

## Prerequisites

Node.js
MongoDB

## Installation

Clone the repository:

```Typescript
git clone https://github.com/your-username/url-shortener.git

```

Navigate to the project directory:

```Typescript
cd url-shortener
```
